在 centos 上安装 git

由于通过 yum 安装的是旧版，所以这里使用源码安装

服务器更新 git 版本报错：
git-compat-util.h:14:2:error “Required C99 support is in a test phase“

解决：使用版本 wget https://github.com/git/git/archive/refs/tags/v2.17.0.tar.gz -O git-2.17.0.tar.gz

Git 命令

init

初始化 git 仓库

中央仓库：可以作为 git 中转仓库，能使用命令仓库提交记录，分支等信息。不能提交代码…

clone

在没有本地仓库的情况下，从远端或本地拉取代码

cherry-pick

挑拣

worktree

支持多个分支同时签出

push

pull

fetch

获取远端 commit（不合并到本地）

add

commit

status

查看当前分支状态：列出文件改动，未跟踪，移除的文件

stash

储存

switch

切换分支

branch

分支

remote

merge

合并分支

rebase

变基：可用户合并多个提交、修改已提交的 commit 的 messaged 等等

修改 commit 信息

  1. 使用命令git rebase -i <commitid>。commitid为要修改的 commit 的前一个 commit,也就是不变的那个 commit
  2. 找到需要修改的 commit，将前面的pick改为reword
  3. 推出当前的编辑器，会打开新的编辑器，提示输入提交信息
  4. 修改信息，wq 保存退出
